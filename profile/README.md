# CodeClub

**"The first rule of CodeClub is: you talk about CodeClub"**

---

## Overview

CodeClub is an invite-only mentorship program where members commit to participating in a structured Agile process to advance their skills.

Each member chooses a course of study and is expected to:

* Follow an industry-standard Agile process
* Set measurable goals
* Continuously improve

---

## Cadence

* **Sprints:** 2 weeks
* **Sprint Commitment:** Minimum ~16 hours/week (ideally more)
* **Capstone Project:**
  Each member completes a capstone project in their chosen domain and presents it to a panel from our network.
