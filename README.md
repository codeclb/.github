## 🧠 What is CodeClub?

**CodeClub** is an invite-only mentorship program designed for engineers who want to move beyond tutorials and into **real-world execution**.

This is not passive learning.

This is:
- 📈 Structured growth  
- 🎯 Measurable outcomes  
- 🔁 Continuous improvement  
- 🤝 Peer + mentor accountability
